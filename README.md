# سیستم مدیریت کتابخانه (Library Management System)

یک سیستم مدیریت کامل کتابخانه با استفاده از PHP، MySQL، HTML، CSS و Bootstrap.

## ویژگی‌ها

- ✅ مدیریت کامل کتاب‌ها (Books)
- ✅ مدیریت مجلات (Journals)
- ✅ مدیریت کتاب‌های الکترونیکی (Ebooks)
- ✅ مدیریت اعضا (Members)
- ✅ سیستم امانت و بازگرداندن (Borrowing)
- ✅ سیستم رزرو (Reservation)
- ✅ مدیریت جریمه‌ها (Fines)
- ✅ داشبورد با آمار کامل
- ✅ رابط کاربری زیبا و ریسپانسیو با Bootstrap 5
- ✅ پشتیبانی کامل از زبان فارسی (RTL)

## نیازمندی‌ها

- PHP 7.4 یا بالاتر
- MySQL 5.7 یا بالاتر (یا MariaDB 10.4+)
- Apache/Nginx Web Server
- مرورگر مدرن با پشتیبانی از JavaScript

## نصب و راه‌اندازی

### 1. کلون کردن پروژه

```bash
git clone [repository-url]
cd cursor
```

### 2. ایجاد دیتابیس

فایل SQL موجود در پروژه را در phpMyAdmin یا MySQL اجرا کنید:

```sql
-- فایل SQL را در phpMyAdmin import کنید
-- یا از خط فرمان:
mysql -u root -p < database.sql
```

### 3. پیکربندی اتصال دیتابیس

فایل `config.example.php` را کپی کنید و نام آن را به `config.php` تغییر دهید:

```bash
# در Windows
copy config.example.php config.php

# در Linux/Mac
cp config.example.php config.php
```

سپس فایل `config.php` را ویرایش کنید و اطلاعات اتصال دیتابیس را تنظیم کنید:

```php
define('DB_HOST', 'localhost');
define('DB_USER', 'root');
define('DB_PASS', '');
define('DB_NAME', 'lms');
```

### 4. اجرای پروژه

پروژه را در وب سرور محلی خود قرار دهید:

- **XAMPP**: پوشه را در `htdocs` کپی کنید
- **WAMP**: پوشه را در `www` کپی کنید
- **Laragon**: پوشه را در `www` کپی کنید

سپس در مرورگر به آدرس زیر بروید:
```
http://localhost/cursor/index.php
```

## استفاده از سیستم

### مدیریت کتاب‌ها

1. از منوی اصلی به بخش "کتاب‌ها" بروید
2. برای افزودن کتاب جدید، روی دکمه "افزودن کتاب جدید" کلیک کنید
3. فرم را پر کنید و ذخیره کنید
4. برای ویرایش یا حذف کتاب، از دکمه‌های عملیات استفاده کنید

### مدیریت اعضا

1. به بخش "اعضا" بروید
2. عضو جدید را اضافه کنید
3. اطلاعات عضویت را مدیریت کنید

### ثبت امانت

1. به بخش "امانت" بروید
2. روی "ثبت امانت جدید" کلیک کنید
3. عضو و آیتم را انتخاب کنید
4. تاریخ سررسید را تنظیم کنید

### مدیریت رزروها

1. به بخش "رزروها" بروید
2. رزرو جدید ثبت کنید
3. وضعیت رزروها را مدیریت کنید

### مدیریت جریمه‌ها

1. به بخش "جریمه‌ها" بروید
2. جریمه‌های پرداخت نشده را مشاهده کنید
3. پرداخت یا بخشش جریمه را انجام دهید

## ویژگی‌های رابط کاربری

- **طراحی مدرن**: استفاده از گرادیانت‌های زیبا و انیمیشن‌های نرم
- **ریسپانسیو**: سازگار با تمام دستگاه‌ها (موبایل، تبلت، دسکتاپ)
- **آیکون‌های Bootstrap Icons**: استفاده از آیکون‌های زیبا و واضح
- **اعتبارسنجی فرم**: اعتبارسنجی سمت کلاینت برای بهبود تجربه کاربری
- **تأیید حذف**: پیام تأیید قبل از حذف داده‌ها
- **هشدارهای خودکار**: پیام‌های موفقیت/خطا به صورت خودکار بسته می‌شوند
